# Sela

A macOS app that translates [ProPresenter](https://renewedvision.com/propresenter/) worship songs from English to Dutch.

## Install

```bash
brew install --cask joehoel/sela/sela
```

Requires macOS 14 (Sonoma) or later on Apple Silicon.

## Disclaimer

This project is **for personal use only**. It is not affiliated with, endorsed by, or connected to Renewed Vision or ProPresenter in any way. It interacts with ProPresenter files through reverse-engineered formats — use at your own risk.
